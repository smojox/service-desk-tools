2. Uploading Your Data

1Upload Process

Follow these steps to upload your CSV file:

  1. Click the upload area or drag and drop your CSV file
  2. Wait for processing - you'll see a loading indicator
  3. Dashboard loads automatically with your data
  4. If upload fails, check file format and try again

CSV file upload process

Click or drag CSV file here

File Format Requirements

  • File must be .csv format
  • First row should contain column headers
  • Required columns: ticketId, subject, status, priority, createdTime
  • Optional columns: companyName, sdm, agent, type, resolvedTime

6. SLA Management

1How to Access SLA Details

  1. Click the "SLA Compliance" metric card on the dashboard
  2. A modal window will open showing detailed SLA breakdown
  3. Use the tabs to view different aspects of SLA data
  4. Search for specific tickets using the search box

SLA management modal with override toggles

Click SLA Compliance card

2SLA Override System

You can manually override SLA status for individual tickets:

Green Toggle

Ticket meets SLA requirements

✓ Compliant

Red Toggle

Ticket breaches SLA requirements

✗ Breach

Important:

Click any toggle to override the system calculation. Changes are saved automatically and persist across sessions.

3Export SLA Data

Click "Export to Excel" button to download current SLA breach data with your overrides included.

7. Generating Reports

1Before Generating Reports

  1. Set your desired filters (Company, SDM, Date Range)
  2. Review the dashboard to ensure data looks correct
  3. Check SLA overrides if needed
  4. Verify charts show the expected data

2How to Generate PDF Reports

Export Buttons

Screenshot: Top toolbar with export buttons highlighted

Click PDF button
  1. Click the "PDF" button in the top toolbar
  2. Wait for generation (usually 10-30 seconds)
  3. Download starts automatically
  4. File saved with descriptive name

Pages 1-3

Pages 4-6

8. Troubleshooting

CSV Upload Issues

Problem: File won't upload

Solutions:

  • Check file is .csv format
  • Verify column headers
  • Try smaller file size
  • Check internet connection

Dashboard Display

Problem: Charts not showing

Solutions:

  • Refresh the browser
  • Check console for errors
  • Clear browser cache
  • Try different browser

PDF Generation

Problem: PDF fails to generate

Solutions:

  • Check console for errors
  • Try smaller date range
  • Refresh page
  • Check internet connection

Filter Issues

Problem: Filters not working

Solutions:

  • Click "Clear Filters"
  • Reload the page
  • Upload data again
  • Try different browser

Need More Help?

If you continue to experience issues:

  1. Check the browser console for detailed error messages
  2. Try with a different browser or device
  3. Contact your system administrator for support

Browser Requirements

Minimum: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

Recommended: Chrome or Edge latest version with 8GB+ RAM
